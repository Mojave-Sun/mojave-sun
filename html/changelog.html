18 September 2020

02 September 2020

ATHATH updated:

  • The message that you receive when you try to use a gang induction package as a mindshielded person no longer implies that your mindshield has given you fake memories of attending an anti-gang seminar.

Arkatos updated:

  • Unrestricted directional access settings will now show correct directions in the related UIs.
  • Theoretical radiuses will now show properly in the Tachyon-Doppler Array UI.

Ryll/Shaps updated:

  • Medbot interaction menus are fixed
  • You can no longer queue multiple medbot flips at once
  • Medbots now have lots more to say, especially if you knock them over!

Shadark updated:

  • Detective gets roundstart normal sunglasses instead of SecHUDs at PubbyStation.
  • Balanced access to arrest option through SecHUDs, now requiring Security access instead of Brig access.
  • Balanced access to portable turrets, now requiring Security access instead of Brig access.

ShizCalev updated:

  • Fixed a scenario where changelings would have missing textures when disguised.
  • Fixed a scenario where camouflage items would have missing textures.
  • Items in your suit storage should have less missing textures now.
  • Restore the ion rifle's unique belt texture.
  • Fixed a few more belt items having missing textures.
  • Fixed washing machines sometimes fucking up an item's worn texture.

tmtmtl30 updated:

  • dynamic families rulesets can now be forced and will execute instead of runtiming
